# OpsTrax Integration Architecture

## Integration Patterns
- Adapter-per-system
- Signed webhook ingestion
- Scheduled sync jobs
- Tenant-aware credential storage
- Mapping tables for external IDs
- Retry and monitoring hooks

## Integration Targets
- GPS / telematics vendors
- ELD providers
- Dashcam systems
- Accounting / ERP
- Payment providers
- Maps and routing APIs
- AI service providers
